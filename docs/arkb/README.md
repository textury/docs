# Arkb User Guide

A guide for getting started deploying web apps and web pages to Arweave's permaweb.

# Installation

Install the latest version. You must have node version 15.40 and above to install [arkb](https://npmjs.org/package/arkb).

```bash
npm install -g arkb
```
